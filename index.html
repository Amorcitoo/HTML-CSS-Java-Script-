HTML
Schau in den Seitenquellentext um den Code dahinter zu verstehen.
In Visual Studio durch das Drücken von ! und Enter erscheint automatisch das Grundgerüst einer HTML Seite.
Im Tag mit Title kommt der Name, der im Tab angezeigt wird.
Im Tag mit Head kommen "Metainformationen" und im Body, dann das, was eigtl. angezeigt werden soll.
Im Head kann ich für CSS eine Stylesektion eröffnen mit dem Tag style.
Ich kann jetzt z.B. zwischen dem Tag mit img und src noch class und einen Namen in Anführungsstrichen einfügen. z.B. HTML-Image.
Mit .HTML-Image kann ich jetzt in der Stylesektion Dinge wie die Breite festlegen, indem ich geschweifte Klammern dahintersetze.
Mit ( < a href " ..... " > angezeigter Name für den Link < / a >" Kann man einen Link einfügen.)
Um z.B. die Überschrift in einer anderen Farbe darzustellen kann ich dem H Tag wieder eine Class zuweisen und diesen Namen dann in der Stylesektion bearbeiten.
Class direkt in die erste Klammer schreiben!
H+1,2,3,4 ist die Headline/Überschrift in verschiedenen Größen
br bedeutet einen Zeilenumbruch
Mit "<....>" öffnet man einen tag und man schließt den tag mit / in der eckigen Klammer
Ein Bild kann ich einfügen indem ich img src="Name des Bildes.Dateityp(z.B. jpg)" als neuen HTML-Tag eröffne, alternativ kann ich auch einen Link zu einem Online verfügbaren Bild einfügen.

Möchte man bestimmte Schriftarten, kann man diese einfach von Google(Google Fonts) importieren. Dafür auf der Seite Schriftart wählen und den entsprechenden Link kopieren
und im Headbereich einfügen.
Danach CSS Code von Google Fonts(unter dem Link) in der stylesektion oben einfügen, damit sich die gesamte Schrift verändert, der tag dazu ist body, aber ohne Punkt davor.
Im selben tag(body) kann man dann auch z.B. Background color, padding(Abstand links, rechts). Schriftgröße etc. festlegen.
Außerdem kann man das Bild mit dem Tag div in einen Container packen und in diesem Container tag eine Klasse(class) für das Bild anlegen, z.B. Image-Container.
Im Style dann wieder einen Image-Container tag eröffnen, background und padding festlegen, da das Bild sonst den Container überdeckt.
